# LeetCode 825 适龄的朋友

看完条件一晃眼...好像叫德摩根定律? (似乎是离散数学里的)

反正展开就对了

为了表达方便修改了运算符

```
  !( b<=0.5*a+7 || b>a || (b>100 && a<100)) 
= !(b<=0.5*a+7) && !(b>a) || !(b>100 && a<100)
= b>0.5*a+7 && b<=a && (b<=100 || a>=100)
= b>0.5*a+7 && b<=a
```

故

```
b ∈ (14, +∞)
a ∈ [b, 2b-14)
```
